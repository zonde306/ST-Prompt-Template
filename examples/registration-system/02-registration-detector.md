# 注册关键词检测条目

## 世界书条目配置说明
- **条目标题（备注）**: `[GENERATE:REGEX:注册|register|登记]`
- **激活状态**: 🔵 启用
- **主要关键字**: 留空（使用正则匹配）
- **条目顺序**: 10

## 条目内容

```javascript
<%
// 检测用户是否提到了注册相关的关键词
const regSystem = getvar('注册系统', { defaults: {} });

// 如果尚未注册且不在注册模式中
if (!regSystem.当前会话用户 && !regSystem.注册模式) {
    // 激活注册模式
    setvar('注册系统.注册模式', true);

    // 提示AI引导用户输入名字
    print(`\n[系统提示：检测到用户想要注册，请引导用户提供他们的名字]\n`);
}
%>
```
